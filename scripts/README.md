# WIAF data pipeline

Парсер запускается через GitHub Actions и обновляет публичные JSON без Google Sheets.

Что обновляется автоматически:
- `public/data/news.json` — новости и сигналы рынка
- `public/data/indices.json` — индексы фрахта и производные ориентиры
- `public/data/fx.json` — валютный фон (USD/RUB, EUR/RUB, CNY/RUB)
- `public/data/rates.json` — ориентиры по ставкам и коридорам
- `public/data/corridors.json` — агрегированный статус коридоров
- `public/data/market_stats.json` — сводка по рынку и активности аукционов
- `public/data/source_status.json` — статус источников и ошибок
- `blog.html` и `news/*.html` — статические SEO-страницы новостей

Логика обновления:
1. Новости тянутся из RSS и HTML-источников.
2. Индексы парсятся из публичных страниц и не обнуляются, если источник временно недоступен.
3. Курсы валют тянутся из XML ЦБ РФ.
4. Ставки и коридоры собираются из текущих ориентиров и автоматически обогащаются сигналами из новостей.
5. При пустом прогоне сохраняется последний валидный набор данных.
